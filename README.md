# employee-task-management
A web-based Employee Task Management System developed using Flask that allows organizations to efficiently assign, track, and manage employee tasks. The system provides a simple and user-friendly interface where tasks can be created, updated, and monitored in real time. It helps improve productivity, transparency, and workflow management within teams.
